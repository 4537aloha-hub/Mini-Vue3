# MiniVue 项目说明

## 项目介绍

本项目是一个 手写简易版 Vue3 内核（MiniVue），用于学习 Vue3 底层原理，核心实现了组件渲染、虚拟DOM（vnode）、h函数、模板解析、组件注册（全局/局部）等核心功能，搭配测试页面验证功能可用性，整体代码精简（约500~800行），适合前端开发者入门 Vue3 源码原理。

项目核心目标：理解 Vue3 渲染流程、组件机制，不追求生产环境可用性，专注于底层原理的极简实现，搭配测试案例（树形组件、组件注册测试等），帮助快速掌握 Vue3 核心逻辑。

## 项目结构

项目目录结构简洁，核心文件如下（按功能分类）：

- 核心内核文件：index.js（入口文件）、parse.js（模板解析）、compile.js（编译处理）、render.js（渲染逻辑）、component.js（组件处理）等，实现 MiniVue 核心功能。

- 测试页面：tree.html（树形组件测试，递归组件实现）、vModel.html（组件注册测试，全局/局部组件验证）等，用于验证内核功能是否正常。

- 配置文件：.gitignore（Git 忽略规则）、README.md（本说明文档）。

## 核心功能实现

本项目聚焦 Vue3 最核心的底层逻辑，主要实现以下功能：

1. h函数（createVNode）：创建虚拟DOM节点，支持渲染普通元素、自定义组件。

2. 组件机制：支持全局组件注册（MiniVue.component）、局部组件定义与使用，实现组件递归渲染（如树形组件）。

3. 模板解析与渲染：解析HTML模板中的组件标签、指令，将虚拟DOM挂载到真实DOM。

4. 响应式基础：简易实现 ref、reactive 等响应式API（根据项目实际内核实现调整）。

## 使用方法

### 1. 环境准备

无需安装额外依赖，纯原生HTML+JavaScript实现，直接通过浏览器打开测试页面即可运行。

### 2. 运行步骤

1. 克隆/下载本项目到本地，确保所有核心文件（index.js、parse.js等）和测试页面在同一目录层级。

2. 打开任意测试页面（如 tree.html、vModel.html），直接用浏览器打开（推荐Chrome、Edge）。

3. 查看页面效果，验证组件渲染、交互功能（如树形组件的展开/折叠、组件内容渲染）是否正常。

4. 查看控制台，可打印组件DOM、vnode等信息，辅助理解底层渲染流程。

### 3. 组件注册示例

#### 全局组件注册（参考测试页面代码）：

```javascript
// 全局注册 divf 组件
MiniVue.component('divf', {
  render() {
    return h('div', '我是全局组件 divf');
  }
});
```

局部组件使用（参考核心示例代码）：

```javascript
// 定义局部组件
const Comp = {
  props: ['foo'],
  render(ctx) {
    return h('div', { class: 'a', id: ctx.bar }, ctx.foo);
  }
};
// 创建组件vnode并渲染
const vnode = h(Comp, { foo: 'foo', bar: 'bar' });
render(vnode, document.body);
```

## 注意事项

- 本项目为 学习用简易实现，不适合生产环境，仅用于理解 Vue3 底层原理。

- 测试页面需确保核心内核文件（index.js）路径正确，否则会报模块导入错误。

- 组件注册时，需先注册（全局/局部）再使用，否则会导致模板解析失败（parse.js 报错）。

- 避免直接操作 MiniVue 接管的DOM，如需修改DOM内容，可通过 setTimeout 等待渲染完成后操作。
